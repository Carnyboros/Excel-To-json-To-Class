How To Use

<caution>
 It has to be  xlsx file.
 If file format is xls(2007yr before format), Use english only.
 You can also use libreOffice Calc (Set format as xlsx).

<Excel to Json>
 Menubar -> Tools-> Excel to Json Converter .
 Set input,output path, Click convert excel files.

<Json to class>
 Create class which matches your json data format. (Be careful lower, upper case)
 Call the JsonToClass.Deserialize();
